## 图像生成推荐

### Seedream 4.5（推荐）

字节最新图像生成模型，支持多种高级功能。

```bash
curl https://api.aigc.it/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <YOUR_TOKEN>" \
  -d '{
    "model": "seedream-4.5",
    "prompt": "一只可爱的猫咪",
    "size": "2K"
  }'
```

**支持功能：**
- ✅ 文生图
- ✅ 图生图
- ✅ 多图融合
- ✅ 批量生成
- ✅ 流式输出
